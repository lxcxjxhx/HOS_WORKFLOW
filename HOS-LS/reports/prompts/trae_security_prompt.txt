## 敏感信息处理
要求：禁止在代码中硬编码密码、密钥等敏感信息，应使用环境变量或安全配置管理。
示例：```python
# 错误做法
db_password = "MySuperSecret123"

# 正确做法
import os
db_password = os.getenv('DB_PASSWORD')
```

## 危险函数调用
要求：避免使用 `eval()`、`exec()` 等可执行任意代码的危险函数，防止代码注入。
示例：```python
# 错误做法
user_input = "__import__('os').system('rm -rf /')"
result = eval(user_input)

# 正确做法
# 使用安全的替代方案，如字典映射或函数调用
```

## 输入验证与过滤
要求：对所有外部输入（如用户输入、API 响应）进行严格的验证、过滤或转义。
示例：```python
# 错误做法
query = f"SELECT * FROM users WHERE name = '{user_input}'"

# 正确做法（使用参数化查询）
import sqlite3
conn = sqlite3.connect('test.db')
cursor = conn.cursor()
cursor.execute("SELECT * FROM users WHERE name = ?", (user_input,))
```

## 文件操作权限
要求：为文件操作设置最小必要权限，并验证文件路径，防止路径遍历攻击。
示例：```python
# 错误做法
with open(user_provided_path, 'r') as f:
    data = f.read()

# 正确做法
import os
base_dir = '/safe/directory'
abs_path = os.path.abspath(os.path.join(base_dir, user_filename))
if not abs_path.startswith(base_dir):
    raise ValueError("非法路径")
with open(abs_path, 'r') as f:
    data = f.read()
```

## 网络请求安全
要求：发起网络请求时使用 HTTPS 并验证 TLS 证书，避免中间人攻击。
示例：```python
# 错误做法（不验证证书）
import urllib.request
response = urllib.request.urlopen('https://example.com/data')

# 正确做法（默认验证证书）
import requests
response = requests.get('https://example.com/data', verify=True)
```
