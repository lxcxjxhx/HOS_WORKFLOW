## 敏感信息保护
要求：禁止在代码中硬编码密钥、令牌等敏感信息，应使用环境变量或安全配置管理。
示例：```python
# 错误做法
API_KEY = "sk-live-123456789"

# 正确做法
import os
API_KEY = os.environ.get("API_KEY")
```

## 危险函数调用
要求：避免直接执行动态生成的代码或命令，防止代码注入攻击。
示例：```python
# 错误做法
user_input = input("Enter command: ")
eval(user_input)

# 正确做法
# 使用安全的替代方案，如预定义函数映射，避免直接执行用户输入。
```

## 输入验证与过滤
要求：对所有外部输入（如用户输入、API 数据）进行严格的验证和清理。
示例：```python
# 错误做法
user_id = request.args.get('id')
query = f"SELECT * FROM users WHERE id = {user_id}"

# 正确做法
import re
user_id = request.args.get('id')
if not re.match(r'^\d+$', user_id):
    raise ValueError("Invalid input")
# 使用参数化查询
```

## 网络请求安全
要求：对外部网络请求使用 HTTPS 等加密协议，并验证 SSL/TLS 证书。
示例：```python
# 错误做法
import urllib.request
urllib.request.urlopen('http://example.com/api/data')

# 正确做法
import requests
response = requests.get('https://example.com/api/data', verify=True) # 默认验证证书
```
